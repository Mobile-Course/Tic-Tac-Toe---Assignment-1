# Tic Tac Toe

A simple two-player Tic Tac Toe game for Android using Kotlin and MVVM architecture.

## Prerequisites

- Android Studio Iguana | 2023.2.1 or newer
- JDK 17 or newer
- Android SDK 34

## How to Run

### Using Android Studio
1. Open Android Studio.
2. Select **Open** and navigate to the project directory.
3. Wait for Gradle sync to complete.
4. Connect an Android device or start an Emulator.
5. Click the **Run** button (green play icon) or press `Shift + F10`.

### Using Command Line
1. Open a terminal in the project root.
2. Run the following command to build and install the debug APK:
   ```bash
   ./gradlew installDebug
   ```
3. Launch the app on your device.

## Features
- 2 Player (Local)
- MVVM Architecture
- "Play Again" functionality
